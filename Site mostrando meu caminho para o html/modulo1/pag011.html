Css externo e interno. Não importa, você só verá a diferença no código do site. Mas vamos explicar:

CSS Inline, Externo e Interno

CSS Inline

O CSS Inline é usado para modificações mais especificas que se utiliza em um determinado elemento ou tag

CSS Externo

O CSS Externo é utilizado caso você queira modificar muitas páginas com o mesmo estilo.
Para isso é precisso crir im link, igual para o favicon, mas ao ênves da palavra favicon, vai CSS junto com o nome do arquivo, como se fosse um link para outra página.

CSS Interno

O CSS Interno é usado na parte cabeça do código, ou seja, na tag Head onde se usa a tag style, seguido pelas tags que você que modificar junto com chaves.
